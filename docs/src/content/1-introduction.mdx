Shakti is a universal component toolkit for React.

## Features
- Declarative props for HTML and CSS functionality
- Intuitive grid system based on flexbox
- Written in TypeScript
- Tree-shaking support
- Built on <a href="https://styled-components.com/" target="_blank" rel="noopener noreferrer">styled components</a>, enforcing DRY object composition
