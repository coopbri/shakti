Shakti contains several components for designing layouts, including a flexbox-based grid system.

## Grid System
The grid system is loosely similar to <DocLink href="https://getbootstrap.com/" target="_blank" rel="noopener noreferrer">Bootstrap</DocLink>'s grid structure:

```tsx
<Grid>
    <Row>
        <Col>ğŸŒ²</Col>
        <Col>ğŸ„</Col>
    </Row>

    <Row>
        <Col>ğŸ</Col>
    </Row>
</Grid>
```

All of the grid components (<code>Grid</code>, <code>Row</code>, and <code>Col</code>) have flex enabled by default, and all <DocLink to="/layout#flex"><code>Flex</code></DocLink> props apply.

### <code>Grid</code>
Serves as a container component for a layout. Rows (<code>Row</code>) are nested within. <code>Grid</code> is flex-enabled with column direction by default.

### <code>Row</code>
Represents a physical row in a layout. Columns (<code>Col</code>) are nested within. <code>Row</code> is flex-enabled with row direction by default.

### <code>Col</code>
Represents a physical column in a layout. Nested within a <code>Row</code>. <code>Col</code> is flex-enabled with column direction by default.

## Other
Aside from the grid system components, there are some other layout components designed to be used independently.

### <code>View</code>
Vanilla <code>div</code> with the ability to augment via <DocLink to="/base">Base</DocLink> props.
