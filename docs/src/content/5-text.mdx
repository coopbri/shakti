A <code>Text</code> component is an augmented <code>p</code> tag with the ability to mix in <DocLink to="/base">Base</DocLink> props as well as the below text props:

<Remark>the below props are simpler overrides for the respective <code><DocLink to="/base#text">Base</DocLink></code> props; they are shortcut props.</Remark>

<PropTable propList={shaktiProps.shortcutText} />

Examples:

```tsx
<Text textCenter>Regular text!</Text>
<Text textCenter bold>Bold text!</Text>
<Text textCenter size={24} border="4px solid deeppink">size 24px text with a border!</Text>
```

```tsx render=true
render(() => (
    <>
        <Text textCenter>Regular text!</Text>
        <Text textCenter bold>Bold text!</Text>
        <Text textCenter size={24} border="4px solid deeppink">size 24px text with a border!</Text>
    </>
))
```
